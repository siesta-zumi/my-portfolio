# My-Portfolio
 今泉康隆ポートフォリオ サイト
 ## 作成への想い
 ポートフォリオ をご覧になった全ての方にエンジニアとして私が今どのような事を学んでいるのかという事だけでなく、私という人物がどのような人物なのかを少しでもイメージしていただきたいという想いを込めて作成しております。
